# Тестовое задание на позицию Backend Developer 
## Испольнитель - Фарфатьянов Денис

### Запустить
```
npm install
docker-compose up/npm run dev
```

### Задание
Написать 2 мини веб сервера и организовать общение между ними с гарантированным контрактом

### Требования
TypeScript, Fastify 

База:
- Fastify
- TypeScript
- Axios
